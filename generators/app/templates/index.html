Sample Client's Visualizations

Fork and Enjoy

At Algorex Health, we often need to compose purpose built webpages to convey visual information. This template provides a simple way to do that without the use of complicated frameworks (it's basically just HTML, CSS, and JS) and can be served as a static site on S3 or other place.

Technologies used:

    Tachyons: is used to help the CSS styling.
    TypeScript: is used rather than javascript because it provides better developer expierence with the typing features. Edit charts.ts to add any visualizations that you need. You can serve the data from static/data
    Rollup: bundles all the code you use in your typescript files so that you load them once.
